# First Contribution Guide

Paste a GitHub issue URL and get a plain English explanation of what it's asking for, 
what skills you need, and exactly how to contribute to it step by step.

## Built with
- Python + Flask
- GitHub REST API
- Groq API (llama-3.3-70b-versatile)

## Run locally
1. Clone the repo
2. Add your GROQ_API_KEY to a .env file
3. pip install flask requests groq python-dotenv markdown
4. python app.py
